#### 1.What is the JSON path used to access the weather description from the Weather API response?  
-The JSON path for accessing the weather description is weather[0].description

#### 2.What is the JSON path used to access the PM2.5 (pm2_5) value from the Air Pollution API response?
-The JSON path for accessing the PM2.5 value is list[0].components.pm2_5
